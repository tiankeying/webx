Unicode 引用

Unicode 是字体在网页端最原始的应用方式，特点是：

  • 支持动态调整图标大小、颜色
  • 默认不支持多色图标
注意：新版 iconfont 支持 SVG symbol 与彩色字体图标引用方式（需开启“彩色”选项并重新生成）

步骤 1：引入 @font-face

@font-face {
  font-family: 'iconfont';
  src: url('iconfont.woff2?t=1745662507504') format('woff2'),
       url('iconfont.woff?t=1745662507504') format('woff'),
       url('iconfont.ttf?t=1745662507504') format('truetype');
}

步骤 2：设置图标样式

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

步骤 3：使用图标

<span class="iconfont">&#xe61d;</span>
  • x
    .icon-x
  • 图片预览右边箭头
    .icon-tupianyulanyoubianjiantou

font-class 引用

相比 Unicode，font-class 更具语义且直观易读。

步骤 1：引入样式

<link rel="stylesheet" href="iconfont.css">

步骤 2：使用图标类名

<span class="iconfont icon-x"></span>
  • x
    #icon-x
  • 图片预览右边箭头
    #icon-tupianyulanyoubianjiantou

Symbol 引用

  • 支持多色图标
  • 支持 font-size 和 color 调整
  • 兼容 IE9+，推荐使用

步骤 1：引入 symbol 脚本

<script src="iconfont.js"></script>

步骤 2：使用 svg 图标

<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-x"></use>
</svg>
